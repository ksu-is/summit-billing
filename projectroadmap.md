# Project Roadmap - Invoice Management System

## Overview

This project is a desktop-based Invoice Management System built using Python and Tkinter. The application allows users to enter customer information, add invoice items, calculate totals, and generate simple invoices.

## Sprint 1 (Planning & Setup)

- [x] Brainstormed project idea based on creating invoices for customers
- [x] Chose Python and Tkinter for the application
- [x] Set up initial project files
- [x] Create basic window layout
- [x] Plan invoice fields and buttons
