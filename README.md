# summit-billing
A program that works as an invoice system designed to help individuals and  small businesses easily create, organize and track payments in a simple and efficient way. 
 ## Features
 - Automatically creates a contract when an invoice is generated
 - Tracks payments and outstanding balances
 - Sends reminders for overdue payments
 - Helps endure services are completed and payment is received

## Purpose
This project aims to simplify business transactions and reducethe risk of unpaid work by combining invoicing, contracts, and payment tracking into one system.
